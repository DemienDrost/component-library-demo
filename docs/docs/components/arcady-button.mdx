## Props

These are props specific to the `arcady-button` component:

|`name`||
|---|---|
|**Description**:|The name of the button, submitted as a pair with the button's value as part of the form data, when that button is used to submit the form.|
|**Type**:|```string```|
|**Default**:|`undefined`|

---

|`variant`||
|---|---|
|**Description**:|Variant of the button - affects it's style|
|**Type**:|```"primary" \| "secondary" \| "ghost"```|
|**Default**:|`"primary"`|

---

|`type`||
|---|---|
|**Description**:|The default behavior of the button|
|**Type**:|```"submit" \| "reset" \| "button"```|
|**Default**:|`"button"`|

---

|`disabled`||
|---|---|
|**Description**:|If the button can be pressed or not|
|**Type**:|```boolean```|
|**Default**:|`false`|
